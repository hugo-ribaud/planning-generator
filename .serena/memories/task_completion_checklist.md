# Task Completion Checklist

## Before Committing
1. [ ] `bun run lint` - Fix all ESLint errors
2. [ ] `bun run build` - Ensure production build works
3. [ ] Test in browser - Verify functionality

## Code Quality
- [ ] French UI strings
- [ ] JSDoc types if adding new data structures
- [ ] Export from barrel files if new component/hook
- [ ] Motion imports from `motion/react`

## Git Workflow
- [ ] Create feature branch from main
- [ ] Meaningful commit messages
- [ ] PR for review if significant changes
